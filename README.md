# 5D Viewer


1) Создайте 3D модель в ревите
2) Создайте ведомость для всех элементов модели ревита
3) Ведомость должна содержать такие столбцы: BUILD_CODE,WORK_TYPE,ID,START,END,COST (смотри пример в файле csv : https://github.com/dronov-dmitry/5D/blob/main/schedule.csv )
4) Заполните эту таблицу, важно заполнить так же айди элементов
5) Экспортируйте 3D модель из ревита в формате fbx
6) Экспортируйте таблицу в формате csv
7) Откройте данное приложение по адресу: dronov-dmitry.github.io/5D/index.html
8) Зарегестрируйтесь
9) Посмотрите видео-инструкцию: https://youtu.be/pL_BOoI8qFA
10) Загрузите свои файлы в два поля на сайте. FBX, CSV (или загрузите тестовые файлы model.fbx, schedule.csv из этой папки)
11) Настройке шаг проигрывания и нажмите Play
12) Смотрите как здание изменяется во времени
