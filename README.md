## Быстрый старт

1. Создайте 3D модель в Revit
2. Создайте ведомость для всех элементов модели Revit
3. Ведомость должна содержать столбцы: BUILD_CODE, WORK_TYPE, ID, START, END, COST (пример: [schedule.csv](https://github.com/dronov-dmitry/5D/blob/main/schedule.csv))
4. Заполните таблицу, в том числе ID элементов
5. Экспортируйте 3D модель из Revit в формате FBX
6. Экспортируйте таблицу в формате CSV
7. Откройте приложение по адресу: dronov-dmitry.github.io/5D/index.html
8. Зарегистрируйтесь
9. Загрузите файлы в два поля: FBX и CSV (или тестовые [model.fbx](https://github.com/dronov-dmitry/5D/blob/main/model.fbx), schedule.csv из этой папки)
10. Настройте шаг проигрывания и нажмите Play
11. Смотрите, как здание изменяется во времени
